Overview
You begin with two types of chips: AND and NOT. These simple built-in chips can be used to create your own custom chips, each
of which can then be used inside of the next one, allowing you to build up layers of increasing complexity

How to Create Chips
New chips can be created from the menu in the bottom-left of the editor.

Input and output pins can be added to the new chip by clicking on the bars on the left and right edges of the editor. These pins will
display their state (0 or 1) as a circle of dark or light color. The state of the input pins can be toggled by clicking on them.

To add previously-created chips into the new chip, simply select them from the bar along the bottom of the editor. These "sub-
chips" can then be connected to the input/output pins of the new chip (or to one another) by clicking on the pins to create wires.

Once you're happy with the chip, you can give it a name and save it from the menu.
You can find all of your created chips (along with a few extra built-in chips) in the Chip Library. From there, you can also delete or
edit your custom chips at any point.

Please Note...
This is a work-in-progress project, which I'm building as part of my video series "Exploring How Computers Work". It is intended
only as a fun toy for experimenting with and learning some basic concepts, and not at all as a serious simulation tool.
The simulator currently has many missing features, which I plan to add in the future. Feedback is greatly appreciated.
